Импорт данных из Excel

Загрузите файл в формате .xlsx со структурой, соответствующей шаблону.

Ручной импорт
Email-импорт (заглушка)

Для тестирования email-импорта положите файл /tmp/email_invoice.xlsx на сервере.

Реальный email-обработчик требует настройки IMAP. После настройки раскомментируйте код в invoice/routes.py.

Требования к файлу

Файл Excel должен содержать следующие колонки:

  • № счета * — уникальный номер
  • № строки * — порядковый номер
  • Наименование изделия *
  • Чертеж / НТД
  • Класс бетона (B25, B30...)
  • Морозостойкость (F200, F300)
  • Водонепроницаемость (W6, W8)
  • Категория поверхности (А6, А7)
  • Отпускная прочность (в %)
  • Объем единицы (м³)
  • Количество (шт)
  • Срок окончания (дата)
  • Заказчик

* — обязательные поля
